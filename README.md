# CTF Writeups

A collection of my Capture The Flag (CTF) competition writeups and walkthroughs.

## About

This repository contains detailed writeups for various CTF challenges I've completed. Each writeup includes the challenge description, my approach to solving it, tools used, and the solution.

## Repository Structure

```
├── README.md
├── [CTF-Fonners-Year]/
│   ├── [Category]/
│   │   ├── [Challenge-Name]/
│   │   │   ├── README.md
│   │   │   ├── files/
│   │   │   └── solution/
```

## Categories

- **Web Exploitation** - Web application vulnerabilities and exploits

## Writeup Template

Each writeup follows this general structure:

1. **Challenge Information** - Name, category, points, description
2. **Initial Analysis** - First observations and reconnaissance
3. **Solution** - Step-by-step walkthrough
4. **Tools Used** - List of tools and scripts
5. **Flag** - The captured flag (if appropriate to share)
6. **Lessons Learned** - Key takeaways and new techniques

## Tools & Resources

Common tools used across challenges:
- `nmap` - Network scanning
- `Burp Suite` - Web proxy and testing
- `Ghidra` / `IDA Pro` - Reverse engineering
- `pwntools` - Binary exploitation
- `Wireshark` - Network analysis
- `John the Ripper` / `hashcat` - Password cracking
